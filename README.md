# Decathlon Maven POM parent

## Use it

To use simple add the following to your pom.xml:

```xml
<parent>
    <groupId>com.decathlon</groupId>
    <artifactId>parent</artifactId>
    <version>$LATEST_VERSION</version>
</parent>
```
